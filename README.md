## 📌 Project Overview

**Resumetrix** is an AI-powered career assistance platform designed to help students, fresh graduates, and professionals optimize their resumes, prepare for interviews, and improve their chances of securing jobs.

The platform combines modern web technologies with Artificial Intelligence to provide:

* ATS Resume Analysis
* Job Description Matching
* Skill Gap Identification
* AI Interview Preparation
* Resume Builder
* Job Application Tracking
* AI Career Assistant (Rexa)

---

## ✨ Key Features

### 📄 ATS Resume Analysis

* Upload and analyze resumes instantly.
* Receive ATS compatibility scores.
* Identify weaknesses and improvement areas.

### 🎯 Job Description Matching

## 🚀 Installation & Setup

### Prerequisites

* Node.js
* npm

### Clone Repository

```bash
git clone https://github.com/bunnybhuvan/Mini-Project.git
cd Mini-Project
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file:

```env
VITE_API_KEY=YOUR_GEMINI_API_KEY
```

### Run Application

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
